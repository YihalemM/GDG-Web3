# ATM Smart Contract - GDG Week 4 Task

A Solidity smart contract simulating an **ATM**.  
Users can **deposit ETH**, **withdraw ETH**, **check balance**, and the **owner can pause/unpause** the contract.

## Features
- Deposit and withdraw ETH
- Check user balance
- Owner-controlled pause/unpause

## Usage
1. Compile `ATM.sol` in [Remix](https://remix.ethereum.org) using Solidity `^0.8.20`.
2. Deploy on **Sepolia** using MetaMask.
3. Use functions: `deposit()`, `withdraw(amount)`, `checkBalance()`, `pause()`, `unpause()`.
